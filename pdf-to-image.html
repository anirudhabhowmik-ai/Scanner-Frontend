Supported Image Formats

  • PNG (Portable Network Graphics): Best for images requiring transparency, sharp text, logos, and screenshots. Lossless compression preserves quality. Supports transparent backgrounds.
  • JPG / JPEG: Best for photographs and complex images where file size matters. Uses lossy compression for smaller files. Ideal for web use.
  • WEBP: Google's modern image format offering superior compression (25-35% smaller than JPG) with both lossy and lossless options. Best balance of quality and file size.

Quality vs. File Size Guide

  • Quality 90-100: Best quality, largest file size. Use for printing, archiving, or when quality is critical.
  • Quality 70-89: Good quality, smaller files. Recommended for presentations, documents, and general use.
  • Quality 50-69: Acceptable quality, much smaller files. Use for web previews, thumbnails, or when bandwidth is limited.
  • Quality below 50: Noticeable compression artifacts. Only use when file size is the primary concern.

DPI Resolution Guide

  • 72 DPI: Standard for web and screen display. Smallest file size. Good for email, social media, and website thumbnails.
  • 150 DPI: Good for presentations, digital documents, and high-quality screen viewing.
  • 300 DPI: Standard for printing. Photos, documents, and professional printing require 300 DPI.
  • 600 DPI: High-resolution printing. Use for detailed graphics, technical drawings, or professional publishing.

PDF to Image vs. PDF to Other Formats: When to Use Which?

  • PDF to Word/Excel: Best when you need to edit text or extract data.
