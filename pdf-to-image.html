Frequently Asked Questions

Which image format should I choose?

PNG: Best for screenshots, diagrams, and images with text. Supports transparency and provides lossless quality. JPG: Best for photos and complex images. Creates smaller files but is lossy (quality degrades with compression). WEBP: Modern format with excellent compression and quality, ideal for web use.

What DPI should I use?

72-150 DPI: Sufficient for web, email, and screen viewing. 200-300 DPI: Standard for home printing and most professional uses. 300-600 DPI: Required for high-quality printing, posters, and professional publications.

How does the quality slider work?

Quality ranges from 1-100. Higher values produce better-looking images with larger file sizes. For most uses, 85-95 provides excellent quality. Use 95-100 for archival or professional work. Lower values (60-80) create smaller files for web use.

Can I convert specific pages only?

Yes! Use the page range field to specify which pages to convert. Examples: "all" for all pages, "1-5" for pages 1 through 5, "1,3,5" for specific pages, or "10-" for page 10 to the end.

What happens with multi-page PDFs?

If converting a single page, you'll receive one image file. For multiple pages, all images are packaged in a ZIP file with descriptive filenames (e.g., document_page_1.png, document_page_2.png).

Are my files secure during conversion?

Yes. All files are processed securely using encrypted connections (HTTPS). Files are automatically deleted from our servers immediately after conversion. We never store, share, or access your documents.

Will the images maintain the original PDF quality?

Yes, when using appropriate settings. At 300 DPI with 95+ quality, images will closely match the original PDF appearance. Higher DPI produces sharper images but increases file size.

Is there a file size limit?

The tool can process PDFs up to 50MB. Processing time increases with file size, number of pages, and selected DPI. Most PDFs convert in under a minute.

Can I convert password-protected PDFs?

Currently, password-protected PDFs cannot be converted. Please remove the password first using our "Unlock PDF" tool, then convert to images.
