    "--outdir", "/tmp",    # Output directory

    --headless is the key. LibreOffice launches in the background, loads the PowerPoint presentation, and exports it as PDF - exactly as if you'd opened PowerPoint and chosen "Save as PDF". All slide layouts, fonts, and images are preserved.

    • Slide transitions - Become static (PDF doesn't support animations)
    • Hyperlinks - Still clickable in the PDF

    📊 PPT vs PPTX - What's the Difference?

    PPT (Older Format)

    Binary format - All data in one binary file
    Used by: PowerPoint 97-2003
    Limitations: Larger file size, less robust to corruption
    LibreOffice handling: Converts perfectly using built-in filters

    PPTX (Newer Format)

    XML-based format - Zip package containing XML files
    Used by: PowerPoint 2007 and later
    Advantages: Smaller files, better for recovery
    LibreOffice handling: Native support, excellent compatibility

    ❓ Technical Questions People Actually Ask

    "Why use LibreOffice instead of a Python library?"

    Python libraries like python-pptx can read PowerPoint content, but they can't render the visual layout - slide masters, exact positioning, font metrics. LibreOffice is the actual PowerPoint rendering engine - what you see in PowerPoint is what you get in PDF. My pitch deck's exact layout would be impossible to replicate with pure Python.

    "What's the difference between PPT and PPTX?"

    PPT is the older binary format (PowerPoint 97-2003). PPTX is the newer XML-based format (PowerPoint 2007+). LibreOffice handles both identically - the conversion pipeline is the same. The professor gets both from students - both work fine.

    "Can I see the LibreOffice command that runs?" 

    soffice --headless --convert-to pdf --outdir /tmp /tmp/12345_pitch_deck.pptx

    That's it. LibreOffice opens, loads the presentation, exports as PDF, and exits. All in the background.

    "What about password-protected PowerPoint files?"

    LibreOffice can't open encrypted files without the password. The tool will return an error. The sales director learned this when he tried to convert a password-protected client deck - had to remove password first.
