    Stuff I Learned From Splitting Thousands of PDFs

    Name Files Immediately

    When you split a PDF, you get files named "page-1.pdf", "page-2.pdf", etc. Rename them right away. I learned this after having 50 files named "page-1.pdf" in my downloads folder and no idea which was which. Be specific - "Contract-Signature-Page.pdf" instead of "page-5.pdf".

    Check Page Numbers Carefully

    Some PDFs have cover pages, tables of contents, or Roman numerals at the start. Page 1 in your viewer might be "page i" in the actual document. Always check what you're actually selecting. I once split out "pages 5-10" and got the table of contents because the numbering was weird.

    Password Protection = Won't Work

    Can't split locked PDFs. Use the Unlock PDF tool first, then split. I get emails about this all the time - "it's not working!" - 90% of the time it's password protected.

    Keep the Original

    Once you split a PDF, you can't easily put it back together. Keep the original file until you're sure you don't need it. I've deleted originals too early and regretted it.

    Huge Files Take Time

    Splitting a 300-page document into individual pages creates 300 files. That's a lot of processing. Be patient. If it's taking more than a few minutes, try splitting into smaller chunks first.

    When Things Go Wrong (And How to Fix Them)

    "The page range isn't working"

    Double-check your syntax. "1-5,7,9-11" is correct. "1 to 5" or "1,2,3,4,5" won't work. Use commas and hyphens only. Also make sure you're not asking for page 50 in a 30-page document.

    "The split is taking forever"

    Big files = big processing time. If you're splitting a 500-page document into individual pages, that's 500 separate files to create. It'll take a few minutes. If it's genuinely stuck, refresh and try splitting smaller batches.

    "I got the wrong pages"

    Happens to everyone. Either you typed the range wrong, or the PDF's page numbering is different from what you expected. Split again with the correct numbers. Keep the original handy.

    "The ZIP file won't open"

    On Windows, right-click and select "Extract All." On Mac, double-click it. If it's corrupted (rare), try the split again. If it keeps happening, email me at nextopdf@gmail.com and I'll help.

    Questions People Actually Ask

    "Can I split a PDF without losing quality?"

    Yes. The pages come out exactly as they went in. Same resolution, same formatting, same everything. Splitting just copies pages - it doesn't recompress or alter them.

    "What's the difference between Split and Extract?"

    Good question. Split breaks one PDF into multiple files. Extract takes specific pages out but leaves the original untouched. Split is for "I want these pages as their own files." Extract is for "I want a new PDF containing just these pages."

    "Do you keep my files?"

    No. Small files never leave your computer. Large files are deleted immediately after processing. I don't have a database of your documents, I don't look at them, I don't sell anything. Your stuff stays yours.

    "Is there a file size limit?"

    For browser processing, under 20MB works best. For larger files, it switches to server mode and can handle up to 100MB. If you have bigger than that, try compressing first or splitting into smaller chunks.

    "Can I split a PDF that's password protected?"

    No. Use the Unlock PDF tool first to remove the password, then split. The tool can't read locked files.

    "Why do I get a ZIP file instead of PDFs?"

    If you split a 50-page PDF, you get 50 separate files. Browsers can't download 50 files at once (they'd ask you to save each one individually). ZIP packages everything together so you get one download with all your pages inside.
