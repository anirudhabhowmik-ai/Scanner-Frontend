    Understanding PDF Encryption Levels

    • No Encryption: Anyone can open, view, print, copy, and edit the PDF without restrictions.
    • User Password Only: Document requires a password to open. Cannot be removed without the password.
    • Owner Password Only: Document opens without a password but has restrictions (no printing, copying, editing). Often removable without a password.
    • Both Passwords: Requires password to open AND has additional restrictions. Both can be removed with the correct user password.
