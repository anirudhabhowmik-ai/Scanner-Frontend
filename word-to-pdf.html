    📄 DOC vs DOCX - What's the Difference?

    DOC (Older Format)

    Binary format - All data in one binary file
    Used by: Word 97-2003
    Limitations: Larger file size, less robust to corruption
    LibreOffice handling: Converts perfectly using built-in filters

    DOCX (Newer Format)

    XML-based format - Zip package containing XML files
    Used by: Word 2007 and later
    Advantages: Smaller files, better for recovery
    LibreOffice handling: Native support, excellent compatibility

    🔤 Font Embedding and PDF/A Standards

    When you convert Word to PDF, fonts can either be embedded or substituted. Here's why embedding matters:

    Font Substitution (Bad)

    If the PDF viewer doesn't have the font you used, it substitutes a similar one. This can change:

    • Line spacing
    • Page breaks
    • Character widths (text can reflow)
    • Overall document length

    My resume looked different because the recruiter's computer didn't have my font - substitution changed everything.

    Font Embedding (Good)

    LibreOffice embeds fonts directly into the PDF. The file includes the actual font data, so it looks identical everywhere. 

    # PDF structure with embedded font
<< /Type /Font
   /Subtype /TrueType
   /BaseFont /ArialMT
   /FontDescriptor << ... >>
   /FirstChar 32
   /LastChar 255
   /Widths [ ... ]  % Character width data
>>

    The freelancer's proposals use a custom-branded font - embedded in the PDF, clients see it perfectly.

    📜 PDF/A for Archiving

    LibreOffice can also export to PDF/A, the archival standard. This ensures documents remain readable for decades. The paralegal uses PDF/A for court filings that need to last.

    ❓ Technical Questions People Actually Ask

    "Why use LibreOffice instead of a Python library?"

    Python libraries like python-docx can read Word content, but they can't render the visual layout - font metrics, paragraph spacing, page breaks, headers/footers. LibreOffice is the actual Word rendering engine - what you see in Word is what you get in PDF. My resume's exact formatting would be impossible to replicate with pure Python.

    "What's the difference between DOC and DOCX?"

    DOC is the older binary format (Word 97-2003). DOCX is the newer XML-based format (Word 2007+). LibreOffice handles both identically - the conversion pipeline is the same. The HR manager's 15-year-old offer letters (DOC) convert just fine.

    "Can I see the LibreOffice command that runs?" 

    soffice --headless --convert-to pdf --outdir /tmp /tmp/12345_resume.docx

    That's it. LibreOffice opens, loads the file, exports as PDF, and exits. All in the background.

    "What about password-protected Word files?"

    LibreOffice can't open encrypted files without the password. The tool will return an error. The paralegal learned this when she tried to convert a password-protected contract - had to remove password first.
